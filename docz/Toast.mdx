# toast

## Basic usage

- 아래와 같은 형식으로 toast를 호출 할 수 있습니다.
```
toast('hello toast!!')
```

<Playground>
  {() => {
    const Test = () => {
      return <Button onClick={() => toast('hello toast!!')}>toast!</Button>
    }

    return <Test />
  }}
</Playground>

## Time

- 노출 시간은 기본 3s 입니다.
- 다음과 같이 시간(ms)을 설정할 수 있습니다.

```
toast('hello toast!!', 1000);
```
