# Toast

- 최상위 컴포넌트 내에 `<Toast />` 컴포넌트가 존재해야 합니다.
```
<div className="App">
  <Toast />
</div>

```

## Basic usage

- 아래와 같은 형식으로 toast를 호출 할 수 있습니다.
```
dispatch(toastActions.toast('hello toast!!'))
```

<Playground>
  {() => {
    const Test = () => {
      const dispatch = useDispatch();
      const openToast = () => dispatch(toastActions.toast('hello toast!!'));

      return <Button onClick={openToast}>alert!</Button>
    }

    return <Test />
  }}
</Playground>

## Time

- 노출 시간은 기본 3s 입니다.
- 다음과 같이 시간(ms)을 설정할 수 있습니다.

```
dispatch(toastActions.toast({ message: 'hello toast!!', time: 1000 }));
```
