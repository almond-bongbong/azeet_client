# alert

## Basic usage

- 아래와 같은 형식으로 alert 팝업을 호출 할 수 있습니다.
```
const openAlert = () => alert('안녕하세요')
```

- alert는 `Promise` 객체를 반환 하므로 팝업이 종료되는 시점을 확인할 수 있습니다.
```
const openAlert = () => alert('하이').then(() => console.log('close'))
```
