# useToast

- toast 팝업을 호출하는 커스텀 hook 입니다.

## Basic Usage

<Playground>
  {() => {
    const Test = () => {
      const toast = useToast();

      return (
        <Button onClick={_ => toast('hello')} text="toast" />
      )
    }

    return (
      <Test />
    )
  }}
