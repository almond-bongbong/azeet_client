# Confirm

- 최상위 컴포넌트 내에 `<Confirm />` 컴포넌트가 존재해야 합니다.
```
<div className="App">
  <Confirm />
</div>

```

## Basic usage

- 아래와 같은 형식으로 confirm 팝업을 호출 할 수 있습니다.
```
dispatch(confirm('정말요?'))
```

- confirm action은 `Promise` 객체를 반환 하므로 파라미터를 통해 선택 값을 알수 있습니다.
- return `true`, `false`
```
dispatch(confirm('정말요?')).then(console.log)
```
