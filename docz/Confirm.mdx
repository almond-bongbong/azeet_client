# confirm

## Basic usage

- 아래와 같은 형식으로 confirm 팝업을 호출 할 수 있습니다.
```
confirm('계속 진행하시겠습니까?');
```

- confirm action은 `Promise` 객체를 반환 하므로 파라미터를 통해 선택 값을 알수 있습니다.
- return `true`, `false`
```
confirm('계속 진행하시겠습니까?').then(console.log)
```
